# Feature: 2 Rich Tui

## Overview
We need a rich TUI to display the output from the agents when in plan mode. The TUI
should be able to display the structured output from the agents in a way that is easy to read and navigate. 
The TUI should also allow for interaction with the agent, such as asking for clarifications or additional information.

## Requirements
- [ ] Display structured output from the agent in a readable format
- [ ] Enable navigation scrolling
- [ ] Allow for interaction with the agent, such as asking for clarifications or additional information

## Constraints
- No constraints

## Acceptance Criteria
- [ ] Plan mode has a TUI that displays the agent's output in a structured and readable format
- [ ] Plan mode TUI allows for navigation and interaction with the agent

## Technical Approach
Should use a popular python TUI.
Should be able to render markdown content in a readable format.
Should be able to support emojis and other formatting to enhance readability.
