## Locked design decisions (implementation input)
- **Category registry in Go** = single source of truth: new `internal/knowledge/category.go` with `Tier` (const TierAlwaysApplied/TierLookedUp) + `Category{Name,Purpose,Boundary,Tier,EntryShape}`, ordered `Categories` list, `AlwaysApplied() []string`, lookup-by-name. Six categories: conventions(always), glossary(always, "term + short gloss"), architecture/gotchas/learnings/decisions(looked-up). Real definition wording is a DELIVERABLE — write real purpose/boundary/shape, not placeholders.
- **Tier declared once**; both search-exclusion and full-load read from registry. Removes the two literal "conventions" hardcodes (search.go skip + set.go reader).
- **Store stays category-agnostic**: remove the conventions SkipDir; `Category` populated by Set.Search (first path segment), not the store. Always-applied categories dropped from search in Set.
- **Hit gains 2 additive fields**: `category` (set by knowledge Set) + `checksum` (SHA-256 raw bytes, computed in store search walk — no second read). JSON additive; existing consumers unaffected.
- **Generalised always-applied reader** in Set returns conventions+glossary tagged by category; keep `knowledge conventions` byte-for-byte for backward compat. 5 templates (plan 02/03/08/13, implement 04) switch to full always-applied set.
- **Checksum EXACT byte-identity** (not fuzzy/SimHash) — load-bearing; rationale documented in docs/knowledge-base.md (lexical≠semantic distance; need is relationship classification, an LLM job; exact is the narrow provable mechanical primitive).
- **No new `knowledge lookup` command**. Dedup = skill lookup branch groups by checksum (deterministic) → consolidation SUB-AGENT merges/precedence(layered, most-specific wins)/surfaces-conflicts → single cited answer. Set.Search does NOT collapse (plain search intact).
- **Contribute routing**: load `knowledge categories`, route by purpose/boundary, steer long content out of glossary. Path = `<category>/<slug>.md`.
- **SKILL.md edits (3.2 + 3.3) edit the same file** — sequence them. Edit the TEMPLATE `templates/skills/workflows/spek-knowledge/SKILL.md`, never the rendered `.claude/skills/...` copy (regenerated by skills.go on init). Use `{{command}}` placeholder in templates.
